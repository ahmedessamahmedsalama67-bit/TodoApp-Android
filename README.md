# TodoApp Android

A simple To-Do List application built with Kotlin and Java.

## Features
- Add tasks easily
- View all tasks in a list
- Data stored using SQLite database

## Technologies Used
- Kotlin (Main Activity)
- Java (Database Helper)
- SQLite (Local Database)
- Android SDK

## How to Use
1. Type a task in the text field
2. Press "Add Task" button
3. Task appears in the list below
